(14.1) Comparitive analysis

  • Horizontal Analysis: also known as trend analysis, this is a technique used to express change over time that has taken place in a series of financial statement data using either amounts or percentages
  • Vertical Analysis:also known as a common size analysis, a technique that expresses each item in a financial statement.
    NOTE:horizontal analysis covers data over a year while vertical analysis only focusses on data within the year.
  • Ratio Analysis involves using ratios as a comparitive tool to evaluate the significance of the significance of the companys financial data, it is more broader than other forms of analysis. There are three types of comparisons,
    • Intracompany
    • Intercompany
    • Industry
  • The ratios being using will be the solvency, liquidity and profitability ratios discussed in earlier chapters.
